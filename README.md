
# M365 E‑commerce (React + Webpack)

A starter e‑commerce front end demonstrating atomic design, a responsive grid, code‑splitting, accessibility hooks, and a modern build process.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173.

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

## Tech
- React 18, React Router
- Webpack 5 with SplitChunks & dynamic imports
- Babel preset‑env & preset‑react
- PostCSS + Autoprefixer
- Jest + @testing-library/react

## Accessibility & Performance
- Semantic elements, labeled controls, keyboard‑friendly navigation, color contrast tokens.
- Images use `loading=lazy` and explicit `width/height` to reduce CLS.
- Route‑level code splitting via `React.lazy`.
